Custom install of Linux
=========================

  Install vim 
  -------------
    * `apt install vim`

  Install Jwm
  -------------
    * `apt install jwm`
    * `apt install xinit`
    * `apt install stterm`
    * `cp /etc/jwm/system.jwmrc .jwmrc`
  
  Try X and Jwm
  ----------------
    * `startx`
    * Open terminal puis setxkbmap fr
